meteo-qt
========

System tray application for weather status information.

Licensed under the GNU General Public License version 3 (GPLv3).

##### Requirements
    python3-lxml
    python3-qt5
    python3-sip
    python3-urllib3

##### Usage
Run ```python3 meteo-qt.py``` from the source folder to get the system tray icon.
